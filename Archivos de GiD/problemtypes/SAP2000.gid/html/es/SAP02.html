Introducción

El tipo de problema GiD SAP2000 es un interfaz entre GiD y el programa SAP2000 de cálculo e estructuras.

Esta interfaz permite a los usuarios de GiD realizar todo el preproceso para la generación de modelos numéricos de geometría compleja y después, pueda exportar el archivo a Sap2000 para su análisis. Una vez ahí, el usuario puede decidir si continua analizando el modelo dentro de Sap2000 o regresar a GiD mediante esta interfaz para visualizar los resultados en GiD. Además esta interfaz implementa una opción para que el usuario pueda generar un archivo de entorno virtual (vrml) con un resultado de postproceso de GiD.

Nota: El programa de cálculo SAP2000 en si mismo no está incluido en esta interfaz, que ha sido testeada con SAP2000 v 15.0.0

La interfaz adapta GiD para permitir:

  • La importación de mallas de SAP2000 en formato de texto .s2k (menu: SAP2000->Importar malla SAP2000...)
  • La creación del fichero ASCII .s2k que es la entrada requerida para ejecutar SAP2000 usando la GUI de GiD que es mas potente que la estándar de SAP2000.

Puede manejare:

  • Modelos 3D de láminas de triángulos o cuadriláteros
  • Modelos 3d de sólidos: hexaedros or tetraedros (que se soportan por SAP2000 como hexaedros degenerados con nodos repetidos)
  • La execución de SAP2000, y mostrar su fichero 'log' (aunque no es posible su ejecución sin ninguna ventana)
  • La conversión del fichero .s2k ASCII de resultados de SAP2000 al formato .post.res de GiD postproceso, para visualizarlo en modo GiD postproceso.
  • La exportación de un resultado en formato VRML, para ser visualizado por ejemplo en un navegador de Internet.
