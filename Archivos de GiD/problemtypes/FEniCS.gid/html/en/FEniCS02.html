Introduction

GiD FEniCS problemtype is an interface between GiD and the FEniCS open source solver.

The FEniCS Project is a collection of free software with an extensive list of features for automated, efficient solution of differential equations.

To download it go to the project page: http://fenicsproject.org

This interface let FEniCS users to graphically handle a complex geometry, define regions, generate a mesh and write all this information as several xml files with the syntax expected by FEniCS

Note: The calculation program FEniCS itself is not included in this problemtype.

The interface version 1.0 is only a first customization step, and can be freely distributed and modified.

FEniCS collaborators are encouraged to enhance this initial version to fit extra solver requirements and to provide this versions to the rest of users.
