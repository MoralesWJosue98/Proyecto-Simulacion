Allows create sets of points, lines or surfaces. These sets are used to configure output file of NASTRAN. NASTRAN's output file only contains output requests for elements or nodes of these sets. To use this condition, the user has to select which kind of request (see table below) should be associated with the set and after click over geometry elements included in the set.
In case that no output set is defined, the output requests will be printed for all elements and nodes.
This condition is used when only part of the model is of interest and not all results in every element or node have to be printed. The necessary time for running the model can be significantly reduced using this condition.
