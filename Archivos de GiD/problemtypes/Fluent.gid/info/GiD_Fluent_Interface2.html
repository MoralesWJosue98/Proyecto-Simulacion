General Information

The GiD interface Fluent permits to extract mesh files suitable for input for the analysis software program Fluent . The procedure to generate these files is the following one:

  • Load the Fluent interface from the "Data->Problem Type" menu
  • Specify the dimension of your problem
  • Specifiy the Boundary conditions of the problem over the geometry entities (this selection can be also performed over mesh elements)
  • Specifiy the Continuum definition of the problem over the geometry entities (this selection can be also performed over mesh elements)
  • Mesh the geometry with the help of all the GiD capabilities (all FLUENT supported elements except polyhedras can be printed in 2D and 3D meshes )
  • Output Fluent file

The interface is fully compatible with Fluent versions from Fluent 4 and GiD 9.1.0b distributions
