Boundary conditions

Boundary conditions can be applied either to geometry (recommended) or mesh entities. 1D or 2D elements can be selected using all the GiD available tools, and grouped into different sets with an unique name. These selections can be performed with the help of the Zones definition window, which can be opened from the FLUENT toolbar or from the data menu:

From the Fluent Toolbar:

From the GiD Menu: Data->Zones Definition

Then the correct Boundary Condition type must be clicked (lines for 2D problems or surfaces for 3D problems)

Each condition group should be related with a single boundary type from the next list:

  • Wall
  • Pressure-Inlet
  • Inlet-Vent
  • Intake-fan
  • Pressure-Outlet
  • Exhaust-Fan
  • Oulet-Vent
  • Symmetry
  • Periodic
  • Shadow
  • Pressure-far-field
  • Velocity-Inlet
  • Periodic
  • Fan
  • Porous-Jump
  • Radiator
  • Mass-flow-inlet
  • Interface
  • Parent
  • Outflow
  • Axis

It's not required to relate each problem's entity with one of these boundary types. By default, GiD-FLUENT interface prints the unassigned edges as Interior Edges (if the edge is surrounded by two elements) or Wall Edges (only one element is connected to the edge).
