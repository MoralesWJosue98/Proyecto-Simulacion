Continuum definition

User can define the fluid and solid domains of each problem with the help of the Continuum condition available either from the FLUENT toolbar or from the GiD menu:

From the Fluent Toolbar:

From the GiD Menu: Data->Zones Definition

Then the correct Continuum entity type must be clicked (surfaces for 2D problems or volumes for 3D problems)

Continuum condition can be applied either to geometry (recommended) or mesh entities. 2D or 3D elements can be selected using all the GiD available tools, and grouped into different sets with an unique name. By default, GiD-FLUENT interface will print all the element of the problem as fluids.
