Prepare the geometry and generate the mesh

The interface supports either 2D or 3D problems. It is important to specify this parameter during the definition of each model (before the zones definition). It can be done in two different ways in the interface:

From the Fluent Toolbar:

From the GiD Menu: Data->Problem Dimension

The geometry is created or imported using the standard GiD commands (See GiD Help for details). The mesh is also generated using standard GiD commands and all GiD element types are supported.

Note: It is recommended to define the zones of our problem before meshing, but this definition can also be performed by selecting mesh elements (see the Boundary Conditions and the Continuum Definition chapters of this help for more details)
