Abaqus : Create Sets

To create a new set with GiD it is necessary to assign a condition to the geometry entities (recommended), before generating the mesh, or to the mesh entities, after generating it.


To create the sets, open the condition window:


Data->Define Sets
Or use the following icon of the Abaqus interface:Â 


Use condition Nset to create nodes sets and the condition Elset to create element sets. Surface sets can also be defined with the help of GiD-Abaqus interface by selecting the surface nodes directly from the screen (SurfaceNodeSet condition). These surfaces can also be defined from 3D element faces (SurfaceSet condition). The choice of the surface definition scheme can affect its behavior during the computation.
