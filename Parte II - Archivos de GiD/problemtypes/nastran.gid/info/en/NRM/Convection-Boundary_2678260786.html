Nastran reference manual : Convection Boundary

MENU SEQUENCE:

Data → Loads → Thermal Loads

This condition defines a convection boundary. It can be assigned to points lines and surfaces. To apply this condition is mandatory to assign a heat boundary condition to define geometric properties of the boundary. By pressing de “Temp…” is possible to define a temperature-dependent convection coefficient. This option can be useful for nonlinear heat transfer analysis.
