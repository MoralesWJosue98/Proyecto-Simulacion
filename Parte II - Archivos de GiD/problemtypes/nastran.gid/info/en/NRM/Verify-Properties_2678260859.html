Nastran reference manual : Verify Properties

MENU SEQUENCE:
Data →  Verify Properties

Checks properties of the model for errors.

Checking features:

          -Finds elements with any property assigned.

          - Checks compatibility of the assigned properties. For example linear mesh elements with beam property assigned has to have       local axes defined.

          - Creates a report list of elements with errors.

Note:

Depending on size of the model, checking the entire model might take few minutes.
