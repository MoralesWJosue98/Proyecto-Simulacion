Nastran reference manual : Importing NASTRAN Models

With GiD-NASTRAN Interface is also possible to import a NASTRAN Model. This feature allows to the user to import all NASTRAN model information. Materials, conditions and problem general data will be imported and loaded in the interface from the NASTRAN model.

After the importing process is possible assign new conditions, unassigned existing ones and create a new NASTRAN model for new simulations.

This feature in the way of total integration of GiD and NASTRAN codes is available from version 2.3beta. 

To import a NASTRAN Model following the steps below:

          1- Load GiD-NASTRAN Interface in GiD:

                   Data → Problem Type → Nastran

          2- Import NASTRAN Model:

              Files → Import → NASTRAN mesh…
