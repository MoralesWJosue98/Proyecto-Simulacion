This problemtype is an example of:
* how to define simple conditions
* evaluation of conditions ( names *_function) over a domain
   look into *.cnd and *.bas
* how to write results directly from the .bas file
* creating a menu and a icon menu bar
* reading an ascii mesh and 
   using the drawopengl tcl command 
   to draw it inside gid
* very simple examples of .bat files to calculate: 
   just rename the .dat file to .res
* print a label from tcl with drawopengl pgffont
* example of a submenu in the icon bar
