How to use

How to use the program:

  • Start GiD and load the FEniCS problemtype from the menu Data->Problemtype->FEniCS, then some conditions and general data are defined.
  • Create or import the geometry, apply the boundary conditions and generate the mesh.
  • Save the model and press calculate to write the .xml input files of FEniCS. Files are saved inside the .gid model folder (now only meshes of tetrahedra and triangles are exported)

If the file .win.bat or unix.bat points to the solver, it will be started (and its log file and error file could be showed in GiD)

  • Results are currently not written or converted to GiD format, then is not possible to directly visualize them changing to GiD postprocess mode.

Note: GiD postproces can import result in common formats, like Vtk, Tecplot or Femap files.

To learn how to use GiD there are several tutorials at www.gidhome.com/support
