Modo de uso

Como usar el programa:

  • Iniciar GiD y cargar el tipo de problema SAP2000 desde el menu Datos->Tipo de problema->SAP2000,

entonces las condiciones y materiales serán definidas y aparecerá un nuevo menu SAP2000.

  • Crear o importar la geometria, aplicar las condiciones de contorno, cargas, materiales y generar la malla. (es posible importar una malla SAP2000, pero no es lo usual)
  • Pulsar Calcular para escribir el fichero .s2k de entrada para SAP2000 y automaticamente arrancará este programa y cargará el fichero mencionado.

(para que arranque SAP2000 su ruta debe ser correcta en el fichero .win.bat , por defecto es "C:\Program Files\Computers and Structures\SAP2000 15\SAP2000.exe")

  • Una vez arrancado SAP2000 es necesario calcular y escribir sus resultados en formato de texte .s2k (desafortunamente ahora no es posible automatizar esto en modo bach)
  • menu: Analize->Run analysis- Run now
  • menu: File->Export->SAP2000 .s2k Text file (seleccionar todo los resultados del analisis para ser escritos)

  • Salir de SAP2000 y volver a GiD para visualizar estos resultados:
  • Se sugiere usar como nombre de fichero <modelo>_res.s2k, en tal caso al salir de SAP2000 este fichero será automaticalmente convertido en <modelo>.post.res

(es posible invocar esta conversión desde el menu de GiD: SAP2000->Convertir resultados SAP2000...)

  • Cambiar a GiD postprocess y los resultados serán leidos automaticament y será posible visualizarlos con todas las posibilidades de GiD.
  • Es posible exportar un resultado escalar en un fichero VRML (Virtual Reality Modelling Language) desde postproceso, desde el menu SAP2000->Exportar fichero VRML...)
